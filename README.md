# OhMyPass 🔒

OhMyPass 是一款为 Obsidian 打造的本地优先、高度安全的密码管理器插件。它让你可以在 Obsidian 内部安全地存储、管理和自动生成密码。

## ✨ 核心特性

- 🛡️ **银行级加密**：基于 `PBKDF2 (600,000轮)` 和 `AES-256-GCM` 算法。主密码仅驻留内存，退出即销毁，密码数据全部以密文形式存储在本地 `data.json` 中。
- 📦 **自动分组卡片**：同名服务多账号自动折叠分组，界面整洁清爽。
- 🔍 **流畅检索**：内置智能防抖搜索和长列表分页渲染，支持千级别密码条目的秒级过滤，拒绝卡顿。
- 🏷️ **多标签 & 树状分类**：支持无限制嵌套标签分类（如 `/工作/内部系统`），一键快速过滤。
- 🔄 **智能导入去重**：支持从主流密码管理器的 CSV 导出文件或本插件加密备份包中导入。导入时自动过滤重复项，冲突密码智能高亮打标，告别数据堆积。
- ⏳ **安全防护机制**：支持无操作超时自动锁定，一键复制密码功能支持设定时间自动清空剪贴板防窥探。
- 🎲 **内置随机密码器**：新建密码时可一键生成可定制长度和字符集的高强度随机密码。

## 📦 安装方法 (手动安装)

1. 从 [Releases](../../releases) 页面下载最新的源文件压缩包，或者克隆本仓库。
2. 确保在包中拥有 `main.js`, `manifest.json`, `styles.css` 这三个文件。
3. 将包含这三个文件的文件夹（如 `oh-mypass`）放入你的 Obsidian 库的插件目录下：
   `你的 Vault 路径/.obsidian/plugins/`
4. 打开 Obsidian 设置 -> 第三方插件 -> 关闭"安全模式"。
5. 在已安装插件列表中找到 `OhMyPass` 并开启。

## 🚀 快速开始

1. **设置主密码**：初次启用插件时，点击右侧边栏的 🔒 图标，按提示设置只属于你的主密码。
2. **添加密码**：通过命令面板搜索 "OhMyPass: 添加新密码" 或在侧栏面板点击"添加"键。
3. **打开密码库**：点击右侧操作栏的锁图标即可进入密码集。界面锁定状态下，点击任意区域即可快速唤起解锁面板。
4. **导入导出**：在面板中使用“导入/导出”，可将过去的 CSV 密码单据一次性合并，自动为您剃除多余数据。

## 🛠️ 开发构建

如果您想自己二次开发这款插件：

```bash
# 1. 安装依赖
npm install

# 2. 编译及自动监听(开发模式)
npm run dev

# 3. 构建生产包
npm run build
```

## ⚠️ 安全须知
- **请务必牢记您的主密码！** 鉴于本地极强的加密策略，一旦主密码遗失，没有任何后门可以找回你的数据。
- 建议定期导出`加密备份文件`或随 Obsidian Vault 一同通过私有同步方案进行备份。
